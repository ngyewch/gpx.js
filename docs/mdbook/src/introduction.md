# @ngyewch/gpx

Javascript GPX parser.

## Overview

This project is an implementation of Topografix's [GPX format](https://www.topografix.com/gpx.asp).

## Usage

```typescript
import {parse} from '@ngyewch/gpx';

const gpx = parse(gpxXmlString);
```
